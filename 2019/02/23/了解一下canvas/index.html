了解一下canvas

描述

<canvas>标签用于绘制图像，但是元素本身并没有绘制能力，而是通过getContext()方法返回一个对象，该对象提供了用于在画布上绘图地方法和属性

浏览器支持

  • IE9+
  • Firefox
  • Opera
  • Safari
  • Chrome

颜色、样式和阴影

  1. fillStyle 设置或返回用于填充绘画地颜色、渐变或模式
  2. strokeStyle 设置或返回用于笔触地颜色、渐变或模式
  3. shadowColor 设置或返回用于阴影地模糊级别
  4. shadowOffsetX 设置返回阴影距形状地水平距离
  5. shadowOffsetY 设置或返回阴影距形状地垂直距离
  6. createLinearGradient() 创建线性渐变（用于画布内容上）
  7. createPattren() 在指定的方向上重复指定的元素
  8. createRadialGradient() 创建放射状/环形的渐变
  9. addColorStop() 规定渐变对象中的颜色和停止位置

线条样式

  1. lineCap 设置或返回线条的结束端点样式
  2. lineJoin 设置或返回两条线相交时，所创建的拐角类型
  3. lineWidth 设置或返回当前的线条宽度
  4. miterLimit 设置或返回最大斜接长度

矩形

  1. rect() 创建矩形
  2. fillRect() 绘制“被填充”的矩形
  3. strokeRect() 绘制矩形（无填充）
  4. clearRect() 在给定的矩形内清除指定的像素

路径

  1. fill() 填充当前绘图
  2. stroke() 绘制已定义的路径
  3. beginPath() 起始一条路径，或者重置当前路径
  4. moveTo() 把路径移动到画布中的指定点，不创建线条
  5. closePath() 创建从当前点回到起始点的路径
  6. lineTo() 添加一个新点，然后在画布中创建从该点到最后指定点的线条
  7. clip() 从原始画布剪切任意形状和尺寸的区域
  8. arc() 创建弧/曲线
  9. arcTo() 创建两切线之间的弧/曲线
  10. isPointInPath() 如果指定的点位于当前路径中，则返回true，否则返回false

转换

  1. scale() 缩放当前绘图至更大或更小
  2. rotate() 旋转当前绘图
  3. translate() 重新映射画布上的 (0,0) 位置
  4. transform() 替换绘图的当前转换矩阵
  5. setTransform() 将当前转换重置为单位矩阵

文本

  1. font 设置或返回文本内容的当前字体属性

  2. textAlign 设置或返回文本内容当前对齐方式

  3. textBaseline 设置或返回在绘制文本时使用的当前文本基线

  4. fillText() 在画布上绘制“被填充”的文本

  5. strokeText() 在画布上绘制文本(无填充)

  6. measureText() 返回包含指定文本宽度的对象

图像绘制

  • drawImage() 向画布绘制图像、画布或视频

像素操作

  1. width 返回ImageData对象的宽度

  2. height 返回ImageData对象的高度

  3. data 返回一个对象，其中包含指定的ImageData对象的图像数据

  4. createImageData() 创建新的，空白的ImageData对象

  5. getImageData() 返回ImageData对象，该对象为画布上指定的矩形复制像素数据

  6. putImageData() 把图像数据返回画布

合成

  1. globalAlpha 设置或返回绘制的当前alpha或透明值

  2. globalCompositeOperation 设置或返回新图像如何绘制到已有的图像上

其他

  1. save() 保存当前环境的状态

  2. restore() 返回之前保存过的路径状态和属性
